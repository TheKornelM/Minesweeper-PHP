(static, constant) MIN_SIZE

Minesweeper class: Stores the data necessary for the game. Fields: - size: The size of the board (size x size). - mineCount: The number of mines in the game, which depends on the generated board size. - remainFields: The number of remaining, unselected fields. If equal to mineCount, the game is won. - flaggedFields: The number of flagged locations. - hasRevealedMine: Indicates whether the player has revealed a mine (true: yes, false: no). - fields: A matrix storing the fields, dynamically allocated.
